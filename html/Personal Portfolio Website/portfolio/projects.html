Contact Management System |
Java, Swing, JDBC, MySQL Developed a desktop application to manage personal contacts using Java Swing for a responsive GUI and JDBC for database connectivity.
Implemented CRUD operations (Create, Read, Update, Delete) to allow users to store and manage contact information persistently in a MySQL database.
Designed a clean UI with a JTable for data visualization and integrated input validation to ensure data integrity (e.g., phone number formatting).

Technical Highlights:


Frontend: Built with Java Swing components (JFrame, JPanel, JTable) using the MVC (Model-View-Controller) design pattern.
Backend: Utilized Core Java for business logic and data processing.
Database: Structured a MySQL schema to store user data efficiently, handled via JDBC for secure and reliable data transactions.
Features: Real-time search functionality, data persistence across sessions, and an intuitive user interface.
